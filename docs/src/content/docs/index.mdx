Wayweft is a TypeScript-first codebase intelligence layer for AI-assisted development. It helps teams inspect change scope, detect refactor drift, surface structural issues, and package repeatable review workflows into portable agent skills.

## What you get today

- A CLI for scanning a workspace or package scope
- Multiple report formats, including SARIF
- Safe-fix scaffolding for low-risk cleanup passes
- Skill bundles for Codex and Claude workflows

## How to use this site

- Start with [Getting Started](/getting-started/) for local setup and first-run commands.
- Use the CLI reference pages for focused command usage.
- Check [Configuration](/config/) and [CI](/ci/) when wiring Wayweft into a larger repo workflow.
- Track planned work in [Roadmap](/roadmap/).
