Wayweft is a TypeScript-first tool for detecting duplicate code, surfacing code quality issues, and preserving codebase memory across AI coding sessions. It helps teams inspect change scope in monorepos, clean up after agent edits, and package repeatable review workflows into portable skill bundles for Claude and Codex.

## What you get today

- **Duplicate code detection** — scan a workspace or monorepo package for duplicate functions, structural drift, and code quality issues
- **Multiple report formats** — text, JSON, and SARIF output for local review or CI pipelines
- **Safe TypeScript codemods** — low-risk cleanup passes with dry-run previews
- **Agent handoff bundles** — portable skill workflows that preserve codebase context for Claude and Codex sessions
