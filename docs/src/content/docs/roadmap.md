## Near term

- expand the command reference as the CLI surface grows
- add deeper examples for monorepos and changed-scope review
- document fix application rules and safety boundaries

## Later

- richer rule documentation
- more examples for agent-oriented review loops
- publishing guidance for common static hosts if deployment patterns start to repeat

This page is intentionally lightweight so the docs can evolve with the CLI instead of inventing a large documentation architecture up front.
